
## Commands

- `new` : 新たにディレクトリを作成し，新規プロジェクトを作成
- `init` : 現在のディレクトリに新規プロジェクトを作成
- `info` : プロジェクトの情報を表示
- `build` : プロジェクトをビルド
- `run` : プロジェクトをビルドして実行
- `oneshot` : プロジェクトを作成せず，単一のファイルに対してコマンドを実行
